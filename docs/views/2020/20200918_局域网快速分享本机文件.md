## python2

```shell
cd $fileName
python -m SimpleHTTPServer
默认端口8000，加参数-p 9090改端口
```

## python3
```shell
cd $fileName
python3 -m http.server
默认端口8000，加参数-p 9090改端口
```

## nodejs
```shell
npm i http-server -g
cd $fileName
http-server
```
