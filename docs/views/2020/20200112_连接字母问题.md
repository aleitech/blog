amatory → am + at + ory



## 为什么被称为连接字母，而不是词素

- 词素是最小的音义结合单位，连接字母不是
- 词源，连接字母曾经有过语义和语法价值
- 失去含义后，形和音的功能凸显

## 连接字母处理

- 并入词干，异形词根。anthropo + logy → anthropology
- 并入词缀，异形词缀。cert + ify → certify
- 保留，独立。phon + o  + gram → phonogram
